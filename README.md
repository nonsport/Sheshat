# Seshat: Advanced OSINT Architecture

Seshat — это высокотехнологичная масштабируемая модульная платформа для проведения разведки по Open Source Intelligence (OSINT). Инструмент сочетает в себе строгий минималистичный дизайн, высокую скорость работы благодаря асинхронности и неограниченную гибкость за счет продвинутой архитектуры автозагрузки модулей.

## 🔥 Основные особенности
* **Minimalist UI**: Строгий консольный интерфейс в белом цвете на базе библиотеки Rich.
* **Modular Architecture**: Легко расширяемый функционал. Просто поместите ваш скрипт в соответствующую папку внутри `modules/`, и он автоматически появится в древовидном меню.
* **Deep Dive Socials**: Поиск по социальным сетям без платных API (продвинутый скрейпинг).
* **Security**: Ротация User-Agent для обхода блокировок.
